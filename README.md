# DiskDataAnalyzer

**Універсальний аналізатор дисків для Windows**

## Опис

DiskDataAnalyzer - потужний інструмент для аналізу та інвентаризації даних на дисках Windows. Автоматично визначає призначення директорій, аналізує архіви, знаходить дублікати та генерує детальні звіти.

## Можливості

### ✅ Базовий аналіз
- Загальна інформація про диск (розмір, використання, вільне місце)
- Топ-20 найбільших директорій
- Аналіз тимчасових файлів
- Розподіл файлів за типами

### ✅ Аналіз архівів
- Підтримка форматів: ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ, RAR, 7Z
- Кількість файлів всередині архіву
- Нестиснутий розмір
- Коефіцієнт стиснення

### ✅ Категоризація директорій
- Автоматичне визначення типу директорії:
  - **project** - проєкти з кодом (.git, package.json, requirements.txt)
  - **backup** - бекапи та архіви
  - **media** - фото, відео, музика
  - **game** - ігри (Steam, Epic, GOG)
  - **document** - документи та робочі файли
  - **system** - системні файли Windows
- Визначення активності (файли змінювались останні 30 днів)
- Рівень впевненості категоризації

### ✅ Пошук дублікатів
- Пошук за MD5 хешем
- Групування однакових файлів
- Розрахунок втраченого місця
- Мінімальний розмір файлу (за замовчуванням 10MB)

### ✅ Пошук великих файлів
- Знаходження найбільших файлів на диску
- Конфігурований мінімальний розмір
- Сортування за розміром
- Обмеження кількості результатів

### ✅ Звіти та експорт
- **Markdown** формат (за замовчуванням)
- **JSON** експорт для програмної обробки
- **CSV** експорт для аналізу в Excel
- **HTML** експорт з стилізованим виглядом
- Детальна статистика по категоріях
- Топ архівів з інформацією про стиснення
- Список дублікатів з шляхами
- Підсумковий звіт для всіх дисків

### ✅ CLI покращення
- Кольоровий вивід (colorama)
- Прогрес-бар для довгих операцій (tqdm)
- Гнучкі параметри експорту

### ✅ GUI інтерфейс (v0.5.0)
- **File Classifier** - класифікація файлів за допомогою LLM
  - Підтримка провайдерів: Claude, OpenAI, Ollama, KiroAI
  - Автоматична категоризація файлів
  - Експорт результатів у CSV/JSON
- **Directory Tree** - візуалізація структури директорій
  - Ієрархічне дерево з іконками
  - Фільтрація за патернами
  - Обмеження глибини
  - Експорт у текстовий формат
- **Folder Compare** - порівняння двох папок
  - Рекурсивне порівняння
  - Порівняння за хешем або розміром/датою
  - Кольорове виділення відмінностей
  - Експорт звіту
- **Junk Detector** - розширене виявлення сміття
  - Тимчасові файли
  - Бекапи та старі бекапи (>30 днів)
  - Дублікати файлів
  - Порожні папки
  - Безпечне видалення з підтвердженням

## Використання

### GUI режим (рекомендовано)
```bash
python src/main.py
```

Відкриється вікно з 4 вкладками:
- **File Classifier** - класифікація файлів з LLM
- **Directory Tree** - візуалізація структури папок
- **Folder Compare** - порівняння двох папок
- **Junk Detector** - пошук та видалення сміття

### CLI режим

#### Аналіз одного диску
```bash
python src/core/analyze_disk.py C:
```

#### Аналіз всіх доступних дисків
```bash
python src/core/analyze_disk.py --all
```

#### Вказати директорію для звітів
```bash
python src/core/analyze_disk.py D: --report-dir O:/my_reports
```

#### Експорт у різні формати
```bash
# JSON експорт
python src/core/analyze_disk.py C: --export json --output report.json

# CSV експорт
python src/core/analyze_disk.py C: --export csv --output report.csv

# HTML експорт
python src/core/analyze_disk.py C: --export html --output report.html
```

## Вимоги

- Python 3.7+
- Windows 7/8/10/11
- Обов'язкові модулі: os, sys, argparse, pathlib, shutil, datetime, collections, hashlib, zipfile, tarfile, json, tkinter
- Опціональні модулі (з graceful fallback):
  - `tqdm` - прогрес-бар
  - `colorama` - кольоровий вивід
  - `rarfile` - підтримка RAR архівів
  - `py7zr` - підтримка 7Z архівів
  - `anthropic` - Claude API для LLM класифікації
  - `openai` - OpenAI API для LLM класифікації

### Встановлення залежностей
```bash
pip install -r requirements.txt
```
