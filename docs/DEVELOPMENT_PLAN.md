## Огляд проєкту

DiskDataAnalyzer - універсальний аналізатор дисків для Windows з можливостями:
- Аналіз використання дисків
- Категоризація директорій
- Аналіз архівів
- Пошук дублікатів
- Генерація детальних звітів
