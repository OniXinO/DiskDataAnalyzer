## [0.2.0] - 2026-04-06

### Додано
- **Підтримка RAR архівів** - аналіз RAR архівів через бібліотеку rarfile
- **Підтримка 7Z архівів** - аналіз 7-Zip архівів через бібліотеку py7zr
- **Пошук великих файлів** - знаходження найбільших файлів з налаштовуваним порогом розміру
- **JSON експортер** - експорт звітів у JSON формат
- **CSV експортер** - експорт звітів у CSV формат для Excel
- **HTML експортер** - експорт звітів у стилізований HTML
- **Прогрес-бар** - індикація прогресу на базі tqdm для довгих операцій
- **Кольоровий вивід** - кольоровий CLI вивід на базі colorama
- **CLI аргументи експорту** - прапорці `--export` та `--output` для вибору формату
- **69 автоматичних тестів** - повне покриття тестами через unittest

### Змінено
- Покращено аналіз архівів для підтримки більше форматів
- Покращено CLI з кращим зворотнім зв'язком користувачу
- Оновлено документацію з новими можливостями

### Виправлено
- Обробка циклічних символічних посилань для запобігання нескінченної рекурсії
- Проблеми з UTF-8 кодуванням в Windows консолі
- Визначення архівів RAR та 7Z

### Технічні деталі
- Graceful fallback для опціональних залежностей (tqdm, colorama, rarfile, py7zr)
- Модульна архітектура з окремими експортерами
- TDD підхід з RED-GREEN-REFACTOR циклом
- Покриття тестами: 69 тестів, 100% успішність

## [0.1.0] - 2026-04-06

### Додано
- Базовий аналіз дисків (розмір, використання, вільне місце)
- Сканування найбільших директорій
- Аналіз тимчасових файлів
- Розподіл файлів за типами
- Аналіз архівів (ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ)
- Розрахунок коефіцієнта стиснення архівів
- Автоматична категоризація директорій:
  - project (проєкти з кодом)
  - backup (бекапи)
  - media (фото, відео, музика)
  - game (ігри)
  - document (документи)
  - system (системні файли)
- Визначення активності директорій (файли змінювались останні 30 днів)
- Пошук дублікатів за MD5 хешем
- Розрахунок втраченого місця через дублікати
- Генерація Markdown звітів
- Підсумковий звіт для всіх дисків
- CLI інтерфейс з аргументами
- Підтримка аналізу одного диску або всіх дисків
- Документація (README.md, CLAUDE.md, DEVELOPMENT_PLAN.md)

### Технічні деталі
- Python 3.7+ сумісність
- Тільки стандартна бібліотека (без зовнішніх залежностей)
- Обробка помилок доступу до файлів
- Оптимізація для великих дисків (обмеження глибини сканування)
- Підтримка WSL1 середовища
