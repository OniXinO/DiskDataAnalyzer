## [0.5.0] - 2026-04-07

### Додано
- **LLM-класифікація файлів** з 4 провайдерами:
  - Claude (Anthropic API)
  - OpenAI (GPT моделі)
  - Ollama (локальні моделі)
  - KiroAI (кастомний провайдер)
- **Плагінна архітектура** з LLMRegistry для розширюваності
- **SQLite-кеш класифікації** з автоматичною інвалідацією при зміні файлів
- **Візуалізація дерева директорій** з:
  - Ієрархічне відображення з іконками (📁 папки, 📄 файли)
  - Фільтрація за патернами (ігнорування .git, __pycache__, node_modules)
  - Налаштовувана глибина
  - Експорт у текстовий формат з Unicode символами
- **Порівняння папок** з:
  - Рекурсивне порівняння директорій
  - Виявлення за хешем (MD5) або розміром/датою
  - Кольорове виділення результатів (ідентичні, різні, тільки в папці 1/2)
  - Експорт звітів порівняння
- **Розширене виявлення сміття** з 5 категоріями:
  - Тимчасові файли (*.tmp, *.temp, *.cache)
  - Файли бекапів (*.bak, *.backup, *~)
  - Старі бекапи (>30 днів)
  - Дублікати файлів (за MD5 хешем)
  - Порожні папки
- **Безпечне видалення** з діалогом підтвердження та попередженням про незворотність
- **4 нові GUI вкладки** інтегровані в головне вікно:
  - Вкладка File Classifier
  - Вкладка Directory Tree
  - Вкладка Folder Compare
  - Вкладка Junk Detector
- **Чеклист мануального GUI тестування** з 100+ тест-кейсами

### Змінено
- Заголовок головного вікна тепер включає версію: "DiskDataAnalyzer v0.5.0"
- Архітектура GUI покращена з навігацією на основі вкладок
- Потокобезпечні оновлення GUI для неблокуючих операцій

### Технічні деталі
- 10 нових core модулів
- 184 автоматичних тести (всі проходять)
- Повне TDD покриття для нових функцій
- Graceful degradation для опціональних LLM залежностей

## [0.1.0] - 2026-04-06

### Додано
- Базовий аналіз дисків (розмір, використання, вільне місце)
- Сканування найбільших директорій
- Аналіз тимчасових файлів
- Розподіл файлів за типами
- Аналіз архівів (ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ)
- Розрахунок коефіцієнта стиснення архівів
- Автоматична категоризація директорій:
  - project (проєкти з кодом)
  - backup (бекапи)
  - media (фото, відео, музика)
  - game (ігри)
  - document (документи)
  - system (системні файли)
- Визначення активності директорій (файли змінювались останні 30 днів)
- Пошук дублікатів за MD5 хешем
- Розрахунок втраченого місця через дублікати
- Генерація Markdown звітів
- Підсумковий звіт для всіх дисків
- CLI інтерфейс з аргументами
- Підтримка аналізу одного диску або всіх дисків
- Документація (README.md, CLAUDE.md, DEVELOPMENT_PLAN.md)

### Технічні деталі
- Python 3.7+ сумісність
- Тільки стандартна бібліотека (без зовнішніх залежностей)
- Обробка помилок доступу до файлів
- Оптимізація для великих дисків (обмеження глибини сканування)
- Підтримка WSL1 середовища
- 69 автоматичних тестів
